css & js trick

css或者js中的小技巧

有时候会发现一些日常不怎么注意但是其实很有意思, 也能排上用场的小技巧, 这里统一记录一下

point-events: none

point-events属性用来控制元素对于鼠标事件的相应, 虽然大部分可选值都是针对svg的, 但是有一个none值是对所有元素都生效的.

完整属性请查看这里MDN: pointer-events

当设定为none的时候元素不会对鼠标事件(点击, 移动等)作出相应, 表示鼠标事件“穿透”该元素并且指定该元素“下面”的任何东西(通过Tab是仍然可以选中的), 鼠标样式也会随着”下面”元素的cursor属性变化

这就能做出一种很炫的效果, 比如”王者荣耀”匹配成功页面上面有一只大鲸鱼游动, 但是依然可以点击到鲸鱼下面的”确认”按钮, 我之前就一直在想如果用web来做该怎么实现这种穿透效果, 那么借助这个属性就可以轻松做到了
